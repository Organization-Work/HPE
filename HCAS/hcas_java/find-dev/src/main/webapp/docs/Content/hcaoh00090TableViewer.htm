Table Viewer

This visualization allows a quantitative display of two parameters simultaneously analogous to a conventional pivot-table. For example, typical EMR data may allow you to select “ICU care unit” on the X axis, and choose “ICD-9 code” on the y-axis. This would produce a two-dimensional table showing the multiple ICU care units along with a display of the most frequent ICD-9 codes. This table is an interactive component, so selecting a column heading such as “MICU” or “(427.31) ATRIAL FIBRILLATION” would add that item to the query filters. Similarly, selecting the cell where these column and row headings meet would add both categories to the current query filter. The table will recalculate automatically when the query filters are modified, or the row or column selections are changed. The Settings button in the upper left can be used to change the number of cells displayed, or sort criteria.
