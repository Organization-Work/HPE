Query Filters

The query filter components indicate which filters are currently being applied to data. This pane allows you to create and refine query parameters including selecting for datatypes (for example, parametrics, text, numeric, and conceptual types), Boolean operations, and filter grouping.  Building query filters is usually an iterative and interactive process where a filter is applied, and the resulting data will be immediately available for examination.  Query filters may be saved for recall or inclusion in subsequent queries at a later date.

Query filters may be added via the Add button at the upper left of this pane. Right clicking on this button allows the user to navigate through various data fields which can be added to the active query filter elements. When a particular data element is chosen, it will appear as a new blank filter element. Typing in the text field will show possible data selections via auto complete. The controls immediately above the filter element allow the application of Boolean filters. The icons to the right of the Boolean selector allow the assignment of a descriptive name, and the ability to save filter set.

Query filters are surrounded by a thin border. All query filters within a particular region are grouped and filtered by the Boolean operator at the top left of the border. To the right of each query filter is an arrow which is used to change filter groupings, and an “X” which is used to inactivate or delete filters.

A number of base filter types are available through MF HCAS configuration. There are specific user interface components to handle:

  • Strings
  • Numeric variables
  • Dates
  • Free text

The base filters may be grouped into units sharing a common operator. Groups may be nested. Available operators include:

  • Any
  • All
  • Not any
  • Not all
  • When

The String/text type filter allows multiple search values. A list of values can either be typed in or copy/pasted in the filter box but each value has to be new line separated. Clicking on the down arrow button enables/disables editing in multi-line mode.
