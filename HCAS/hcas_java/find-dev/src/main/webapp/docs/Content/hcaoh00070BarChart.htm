Bar Chart

 

This visualization displays values for a single data dimension. The selected parameter is highlighted in the list of parameters in the right-hand column. The bar graphs in the central portion of the display indicate the tag names for that parameter and the number of records which are tagged in each category. Visualization is interactive, so that clicking on a particular segment adds that category to the active query filters in the left hand pane.

The top-most bar on the display is a stacked bar chart which indicates the number of tag occurrences within the result set. This bar simultaneously indicates the sum of all tags (up to the number of tags shown in the user interface; default = 20) and the total number of results. This instantly identifies whether untagged records exist, or whether the total number of tag occurrences is greater than the number of records in the result set.

The settings button at the top of the pane may be used to change the number of visible tags, alter the sort order, or display percentages instead of absolute tag counts.

In addition to the graphical data representation, the drop-down menu at the upper left of the Bar Chart display shows a list of categories for the selected parameter. This is particularly useful for seeing more detail than the graphic representation display, such as a larger number of categories. Data may be sorted numerically or alphabetically, and multiple categories may be selected simultaneously via the check boxes next to the category name. After selecting categories, these may be added to the active query filter using the Add Filters button at the upper left of this pane.
