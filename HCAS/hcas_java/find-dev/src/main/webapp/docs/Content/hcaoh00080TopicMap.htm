Topic Map

This visualization is a multidimensional data display. Parametric categories are indicated by a veil which may be clicked to uncover specific data elements belonging to this category. Data elements are organized by frequency of occurrence with the most prevalent values being in the upper left corner of a region, and the least prevalent values placed in the lower right corner. Frequency of occurrence is also indicated by the size of each polygon. Each node is labeled with a dated description as space permits. Dated description labels and record counts may also be seen when the cursor hovers over a particular shape. Visualization is active, so that clicking on a particular node adds it to the currently selected query filter.

Categories for visual display are selected using the Select Fields drop-down at the upper right area of this pane. Clicking on this element displays all of the possible parametric values which may be visualized. The checkbox to the right of each specific category is used to select or deselect an item. The checkbox at the heading for this list can be used to select/deselect whole groups of categories at one time. The Refresh button associated with this control should be pressed after category selection to update the display. Because this visualization conveys multiple dimensions on one screen, best results are obtained by selecting a limited number of categories for display at one time, to ensure adequate screen space for visual display.

The topic map has two sliders at the top of the pane. The “categories” slider selects the total number of categories which are shown on screen, and may be varied between 50 and 300. This control can be used to increase or decrease the number of elements on-screen as desired. The “discard” slider controls how many high-level categories are discarded, before categories are used for the visualization. The default is zero, however, this control can be used when a few categories are dominating the visual field of the display.

The topic map also allows for comparison between two query cohorts. This is enabled by using the Select for Comparison button when the first query is on-screen. At this point, a new query is performed by modifying the query filters in the left hand pane as desired. When the new cohort is displayed, the Compare button is then pressed, and a comparison is shown.

As in the normal topic map view, multiple data dimensions are simultaneously represented on-screen. However, each on-screen node now represents the correlation with either the first query, or the second query. The strength of correlation is indicated both by polygon size (running from largest in the upper left region to smallest in lower right corner) and also by color brightness. The direction of correlation is indicated by the color, where correlations with the first query are denoted in red, and those correlating with the second query are colored green.
