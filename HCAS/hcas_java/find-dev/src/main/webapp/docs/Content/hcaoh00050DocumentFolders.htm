Folders

Folders allow you to store and recall particular records of interest in a hierarchical way.

Use the Add button at the upper right of this pane to create a new folder in the root location. If you would like, you may also save an optional associated description and share it with a group and mark it read-only for restricted access.

Folders, Document Lists and Saved Queries are shown in the lower portion of the above panel.

Folder

A folder can have one or more of the following as its content:

  • Folder (sub-folder or child folder)
  • Document List
  • Saved Query (can be created from the Filters tab)

The menu options for folders are accessible by clicking in the right arrow and are listed below:

  • Add - allows to add another folder or list
  • Delete - deletes the folder
  • Edit - allows editing of the folder name, description and sharing
  • Misc Action - allows cut, copy and paste operations


Document List

A Document List (similar to a file) has a list of records stored which can be recalled later

You can create a Document List using the folder menu option Add (see Folder section)

The check-box immediately to the left of document list indicates whether the list will be available to be used when reviewing records in the document view (see Results).

The menu options for Document Lists are accessible by clicking in the right arrow button:

  • Import - allows loading the list from an associated file
  • Export - allows saving the list to an associated file
  • Edit - allows editing of the folder name, description, shared groups and the list of values
  • Delete - deletes the document list
  • Add Filter - uses the records in the document folder as the base set of records being reviewed
  • Misc Action - allows cut, copy and paste
