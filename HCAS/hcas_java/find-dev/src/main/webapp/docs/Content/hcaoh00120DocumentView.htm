Document View

This view shows the complete data/information for a particular record in the result cohort. The panel on the left side indicates record data which may be a combination of structured, semi-structured, and unstructured elements. Tabs across the top of the left panel change the data display to their respective data elements. The right panel includes document folders (upper), turning query filters (middle), and relevant concepts (lower).

The document folder allows inclusion/exclusion of the record from a particular document list. If no lists are shown, then it is possible new document lists have been chosen to be visible (see Document Folders).

Clicking a particular query filter will highlight data in the left panel to indicate why this record matched the current search criteria.

The relevant concepts show metadata associated with this record. Clicking a particular concept will highlight data in the left panel to indicate where this concept was noted.

Navigation controls in the upper right corner of the document view allow quick access to prior/next data, as well as prior/next records.
